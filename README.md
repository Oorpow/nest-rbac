# 1. nestjs-RBAC简易案例

文章：https://juejin.cn/post/7212526902773661752

# 1. 项目启动

```bash
pnpm install
pnpm start:dev
```

# 2. 创建.env文件

```
MYSQL_HOST='localhost'
MYSQL_PORT=3306
MYSQL_USERNAME='xxx'
MYSQL_PASSWORD='xxx'
# 要连接的数据库名称
MYSQL_DATABASE='database name'
```
